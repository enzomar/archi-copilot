# Glossary - Sample E-Commerce

| Term | Definition |
|------|------------|
| API Gateway | Entry point for all client requests, handles routing, authentication, and rate limiting |
| Aurora | Amazon's MySQL/PostgreSQL-compatible managed database service |
| Bounded Context | A logical boundary within which a particular domain model applies (DDD concept) |
| CAB | Change Advisory Board - reviews and approves production changes |
| CDC | Change Data Capture - technique for tracking database changes for synchronization |
| EKS | Elastic Kubernetes Service - AWS managed Kubernetes |
| Event Sourcing | Storing state changes as a sequence of events rather than current state |
| GitOps | Using Git as single source of truth for declarative infrastructure |
| Microservices | Architectural style where application is composed of small, independent services |
| PCI-DSS | Payment Card Industry Data Security Standard |
| PIM | Product Information Management system |
| Saga | Pattern for managing distributed transactions across microservices |
| Service Mesh | Infrastructure layer for service-to-service communication (e.g., Istio) |
| SLA | Service Level Agreement - committed uptime and performance targets |
| Strangler Fig | Migration pattern that gradually replaces legacy system functionality |
