# Sample E-Commerce Platform - Architecture Context

## Overview

This is a sample e-commerce platform modernization project. The organization is transitioning from a legacy monolithic architecture to a modern microservices-based platform. The project aims to improve scalability, reduce time-to-market for new features, and enhance the overall customer experience.

## Technical Stack

**Current State:**
- Java 8 monolith on WebLogic
- Oracle 12c database
- On-premise data center

**Target State:**
- Java 17 / Spring Boot microservices
- PostgreSQL + Redis
- Kubernetes on AWS
